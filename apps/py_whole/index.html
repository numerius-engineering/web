    Py-Whole Help

    Getting Started

    Py-Whole is a browser-based offline notebook workspace. Open or create a notebook, write Python or markdown cells, and run code directly in the embedded Pyodide runtime.

    • `Shift+Enter` runs the active cell and advances.
    • `Ctrl+S` or `Cmd+S` saves the current notebook and workspace state.
    • Use the `Packages` panel to load bundled packages beyond the default runtime set.
    • Use `Project Files` for notebooks, text files, and import/export workflows.

    Runtime Notes

    Runtime execution is fully local inside the browser. Large packages are bundled into the single-file release but are loaded into the Python environment only when you request them.

    • The first runtime start may take longer while the embedded Python environment initializes.
    • Interrupt requests are cooperative and may not stop execution in the middle of every operation.
    • The experimental debug mode is intended for lightweight inspection and breakpoint-driven stepping, not full IDE-class debugging.
    • Use `Run Runtime Check` to verify package availability and the local matplotlib backend on this device.

    Generic Privacy Policy

    Py-Whole is designed to operate locally in your browser and does not collect, transmit, sell, or share user data with Numerius Engineering LLC.

    • No account sign-in is required.
    • No analytics, telemetry, or remote usage tracking are built into this offline application.
    • Notebook content, workspace files, and saved state remain on the local device unless you explicitly export or share them.
    • If you host this file yourself, any server logging or network policy is controlled by that hosting environment, not by the application logic.
