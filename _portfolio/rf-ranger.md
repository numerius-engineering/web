<section class="page-header">
  <p class="eyebrow">Portfolio</p>
  <h1>RF Ranger</h1>
  <p class="lead">RF Ranger is a browser-based RF engineering workspace for modeling radios, antennas, sites, and propagation paths, then turning those inputs into link analyses, reports, and portable exports from a standalone web application.</p>
</section>

<section class="section-grid">
  <article class="content-block">
    <h2>Project focus</h2>
    <p>This project centered on turning a technically dense RF workflow into a browser-delivered application: map-based site management, configurable radios and antennas, propagation-model selection, simulation output, and a release path that preserves portability in a single HTML file.</p>
    <p><a href="{{ '/apps/RF_Ranger_v0_66_0_single_file.html' | relative_url }}">Launch the live browser app</a></p>
  </article>
  <article class="content-block">
    <h2>What it demonstrates</h2>
    <p>RF Ranger demonstrates Numerius Engineering's ability to build domain-specific tools for engineering users, expose complex modeling options through a structured UI, and package a modern frontend application into a distribution-friendly standalone artifact.</p>
